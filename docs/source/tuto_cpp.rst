C++ tutorial
============

Installation
------------

To use Sdot, you can install it inside the julia command-line:

.. code-block:: console

   $ using Pkg
   $ Pkg.add( "Sdot" )

A simple problem
----------------

For example:

.. code-block:: CPP
   
   #include <sdot/sdot.h>
   using namespace sdot;
   
   int main() {
   }
